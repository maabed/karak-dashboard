Default Style

In order to use blue dashboard customized inputs the input should be wrapped with an .input-field class and the inputitself should have an id that is the same as its label.

Login Style

Login With Icons

Using icons with input fields might be necessary sometimes. In order to use those Icons the input should be wrapped with .input-field class and preceded by the desired icon.

Ranges

Ranges are used by defining an HTML input but gets initialized by the Javascript



Slide Toggle

To use the customized slide toggle you need add the .ch-toggle in addition to color class .ch-toggle-green

Dropdown select

Customized dropdown select can be used by adding .select2-init-single class to be initialized by Javascript for the default dropdown or another set of classes such as .select2-init-multiple

Checkbox Controls
Color Options

In order to use our customized checkboxes you need to give the checkbox a label and wrap them with the .checkbox class.

Shape Options

To give the checkbox a circular shape you need to give the wrapper of the checkboxes a .checkbox-circle class.

Yes/No checkboxes

To use the yes-no checkbox you need to wrap the input with .yes-no-check class and give the checkbox field a unique id to be initialized in the Javascript.

Date Controls
Range Date Picker

You can use our customized Range Pickers by adding an input element and giving it a unique name to be initialized in the Javascript.

 
Simple Date Picker

If you needed a simple date picker you can define an input element and give it a unique name be initialized in the Javascript.

Advanced Datepicker

As for the advanced date picker you define a div element and give it a class attribute to initialize it in the Javascript.

Tag Input

To use the Tag Inputs you need to add the data-role attribute to determine which style you want to use.

Drag N' Drop

In order to use the drag and drop area you need to define a form and give it a unique id to initalize it in the Javascriptin addition to the class dropzone.

Text Editor

All you need to do is adding a unique id to a div element and initalize it in the Javascript.
